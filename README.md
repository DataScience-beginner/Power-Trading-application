# Power Trading Application

## Overview
This project is a standalone web application designed for power trading. It features two separate interfaces:

1. **Founder and Supplier Interface**: Provides global insights, client management, and advanced analytics.
2. **Client-Facing Interface**: Offers personalized dashboards and reports for individual clients.

## Project Structure
```
Power-Trading-application/
│
├── backend/        # Backend service code
├── frontend/       # Frontend application code
├── .gitignore      # Ignored files and directories
└── README.md       # Project documentation
```

## Getting Started
1. Clone the repository.
2. Follow the setup instructions for the backend and frontend.
3. Deploy the application to Railway.
