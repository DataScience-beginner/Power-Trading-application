# Security Compliance Checklist

- [ ] Secret scan completed.
- [ ] Tracked artifact hygiene passed.
- [ ] Python dependency scan completed.
- [ ] Frontend dependency scan completed.
- [ ] Workflow permissions are least privilege.
- [ ] Production environment secrets are protected.
- [ ] Tenant isolation tests passed.
- [ ] TOTP MFA and secure-cookie tests passed.
- [ ] WAF verification, rate-limit, quota, request-ID, and alert contracts passed.
- [ ] Enterprise identity status exposes no secrets.
- [ ] Security audit export is administrator-only and bounded.
- [ ] Control evidence generator passed.
- [ ] Provider-dependent gaps are named in the roadmap.
- [ ] Findings have owners and due dates.
- [ ] Exceptions have explicit approval.
