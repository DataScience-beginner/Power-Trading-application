# Release Management Checklist

- [ ] PR and reviewers recorded.
- [ ] Tested commit SHA equals release SHA.
- [ ] DEV passed.
- [ ] TEST passed.
- [ ] UAT result recorded.
- [ ] Migration and backup status recorded.
- [ ] Cofounder production approval recorded.
- [ ] Rollback target recorded.
- [ ] Post-deploy health check passed.
- [ ] Evidence stored with the release.
