# Data Quality Checklist

- Client/portfolio/date scope validated.
- Policy and engine versions recorded.
- Synthetic/actual classification explicit.
- Coverage, duplicates, interval count, timestamp, quantity, and rate rules evaluated as configured.
- Every finding includes evidence and recommended action.
- No source record, transaction, schedule, or calculation modified.
- High findings escalated before downstream decision use.
- Tests include valid, missing, duplicate, malformed, and cross-tenant cases.
