# Vendor Risk Review Template

For each critical provider, record owner, service, data handled, hosting regions, subprocessors, encryption, access controls, incident notification, availability commitment, backup/exit plan, assurance reports, contract renewal, residual risk, and approval.

Current provider categories include Railway, GitHub, email/SMS delivery, AI model providers, weather data, WAF/CDN, identity provider, object storage, monitoring/SIEM, and malware scanning.

Do not store vendor API keys, credentials, contracts, or confidential assurance reports in this repository.
